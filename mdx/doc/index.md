# 快速开始

#### 对 Material UI 的简单业务组件封装

目前支持：

- EasyDataGrid 表格组件
- EasyToolBar 表格工具栏/搜索栏
- EasyUpload 文件上传组件
- EasyForm 简单表单组件
- EasyDetail 详情组件
- EasyModal 模态框组件
- EasyCard 卡片组件
- EasyButtonList 按钮列表

```
npm install easy-material
yarn add easy-material
```
